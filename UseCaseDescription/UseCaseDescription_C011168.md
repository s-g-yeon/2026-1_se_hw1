# 설문 등록

| no. | actor action | system response |
| --- | --- | --- |
| 1 | 설문 등록 버튼을 누른다. | - |
| 2 | - | 설문 등록 화면을 출력한다. |
| 3 | 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 입력한다. | - |
| 4 | 등록 버튼을 누른다. | - |
| 5 | - | 입력 정보를 저장하고 등록 완료 메시지를 출력한다. |

# 응답 통계 정보 조회

| no. | actor action | system response |
| --- | --- | --- |
| 1 | 통계 조회 화면으로 이동한다. | - |
